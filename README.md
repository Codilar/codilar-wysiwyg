# Codilar Advanced wysiwyg Editor

`Codilar Wysiwyg` is a Magento Extension for CMS Pages 

## Development

It's a Magento Extension used with [GrapesJs](https://grapesjs.com/)'s Free and Open Source Web Builder Framework.

## Installation

Create Name namespace of `Codilar_Wysiwyg` in your Store and copy all files from this repository
